Orientação a Objetos

Dica Extra: Trabalhando com Lista e classe ArrayList

Em Java, para armazenar múltiplos objetos relacionados (como várias avaliações para uma pessoa), usamos listas. O tipo mais comum é o ArrayList.

A classe ArrayList é um tipo de lista (estrutura linear estudada que vocês viram em Estrutura de Dados) que permite a manipulação dinâmica de elementos. Podemos criar listas que armazenam objetos de qualquer tipo.

import java.util.ArrayList;
import java.util.List;

List itens = new ArrayList<>();

itens.add("Ana");
itens.add("João");
itens.add(10); // permitido, pois é uma lista "não tipada"

System.out.println("Itens cadastrados: " + itens);

Lista com tipo definido (genérica)

É possível definir o tipo de elementos que a lista deve aceitar:

List<String> itens = new ArrayList<>();
itens.add("Ana");
itens.add("João");
// itens.add(10); // ❌ erro de compilação

Usar listas genéricas aumenta a segurança e clareza do código, evitando inserções indesejadas.

Métodos mais comuns da classe ArrayList

Método Descrição
add(elemento) Adiciona um elemento ao final da lista.
get(indice) Retorna o elemento que está na posição indicada.
remove(indice) Remove o elemento da posição indicada.
remove(elemento) Remove a primeira ocorrência do objeto na lista.
size() Retorna a quantidade de elementos na lista.
isEmpty() Retorna true se a lista estiver vazia.
contains(elemento) Retorna true se a lista contiver o elemento especificado.
clear() Remove todos os elementos da lista.

Exemplo com objetos personalizados

List<String> nomes = new ArrayList<>();

nomes.add("Ana");
nomes.add("Juka");

System.out.println("Nomes cadastrados: " + nomes);
System.out.println("Total de nomes cadastrados: " + nomes.size());
// removendo um elemento
nomes.remove("Ana");
System.out.println("Nomes cadastrados: " + nomes);
System.out.println("Total de nomes cadastrados: " + nomes.size());
