Passo 8: Anotação JPA na Classe Carta

Agora vamos configurar a classe Carta para que seja persistida no banco de dados com as anotações do JPA (Java Persistence API).

📦 Local:

No pacote model, edite a classe Carta.java com as anotações conforme instruções abaixo.

🎯 Requisitos para a entidade:

  • Nome da tabela no banco: tbl_carta
  • Atributo id como chave primária com geração automática (GenerationType.IDENTITY), nome no banco: car_id

Considere: “nome atributo na classe → nome do atributo mapeado no BD”

  • Atributos obrigatórios:

    • nome → car_nome
    • ataque → car_ataque
    • defesa → car_defesa

  • Atributos opcionais:

    • categoria (Enum Categoria) → car_categoria
    • raridade (int) → car_raridade

Para o atributo categoria, mapear como texto (não índice):
@Enumerated(EnumType.STRING)

Enum Categoria:

⚠️ O enum não precisa de anotações JPA. Ele será interpretado corretamente com @Enumerated(EnumType.STRING) na classe Carta.
