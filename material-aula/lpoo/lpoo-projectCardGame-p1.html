Objetivo desse Tutorial (Parte 1)

Nesta primeira aula do projeto LPOO Card Games, realizaremos alguns passos. Nessa primeira parte realizaremos as configurações iniciais do projeto:

  • Criação do projeto no NetBeans utilizando Maven;
  • Organização da estrutura do projeto em pacotes (MVC);
  • Elaboração das classes Jogador, Baralho e Carta;
  • Criação do banco de dados;
  • Inclusão das dependências no pom.xml;

Etapas parte 1

Passo 1: Criação do Projeto no NetBeans

  1. Abra o NetBeans.
  2. Vá em File > New Project.
  3. Selecione Java with Maven > Java Application.
  4. Configure:
    • Project Name: LPOO-CardGame
    • Group Id: br.edu.seuprojeto
    • Artifact Id: lpoo-cardgame
  5. Finalize clicando em Finish.

Passo 5: Configuração do arquivo pom.xml

Localize o arquivo pom.xml na pasta Project Files. Nesse arquivo vamos adicionar as dependências do JPA, Hibernate e JDBC. Abaixo da tag <properties> inclua a tag <dependencies> :

<dependencies>
        <!-- Hibernate 5 como provedor JPA -->
        <dependency>
            <groupId>org.hibernate</groupId>
            <artifactId>hibernate-core</artifactId>
            <version>5.6.15.Final</version>
        </dependency>
        <!-- JUnit para testes -->
        <dependency>
            <groupId>junit</groupId>
            <artifactId>junit</artifactId>
            <scope>test</scope>
            <version>4.13.2</version>
        </dependency>
        <!-- Driver PostgreSQL -->
        <dependency>
            <groupId>org.postgresql</groupId>
            <artifactId>postgresql</artifactId>
            <version>42.2.24</version>
        </dependency>
        
        <!-- JPA API (javax.persistence) -->
        <dependency>
            <groupId>javax.persistence</groupId>
            <artifactId>javax.persistence-api</artifactId>
            <version>2.2</version>
        </dependency>
        <!-- JAXB (necessário para Java 11+ e compatível com Java 21) -->
        <dependency>
            <groupId>javax.xml.bind</groupId>
            <artifactId>jaxb-api</artifactId>
            <version>2.3.0</version>
        </dependency>
        
    </dependencies>
